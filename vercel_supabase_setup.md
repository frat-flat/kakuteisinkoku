# データベース(Supabase)連携のための設定手順

データベース（Supabase）とスプレッドシートの両方にデータを保存し、確実に動作させるための設定手順です。
この設定を行えば、データの二重バックアップが可能になり、より安全に運用できます。

## 手順 1: Supabaseの接続情報を確認する

1. [Supabase](https://supabase.com/) のダッシュボードにログインし、対象のプロジェクトを開きます。
2. 左メニューの **Settings (歯車アイコン)** > **API** をクリックします。
3. 以下の2つの情報をコピーして控えておきます。
   - **Project URL** (例: `https://xxxxxxxx.supabase.co`)
   - **Project API keys** の `anon` / `public` キー (例: `eyJh...`)

## 手順 2: Vercelに環境変数を設定する

1. [Vercel](https://vercel.com/) のダッシュボードで、対象のプロジェクト (`kakuteisinkoku`) を開きます。
2. 上部タブの **Settings** をクリックします。
3. 左メニューの **Environment Variables** をクリックします。
4. 以下の2つの変数を追加します。

| Key (変数名) | Value (値) |
|---|---|
| `SUPABASE_URL` | 手順1で控えた **Project URL** |
| `SUPABASE_ANON_KEY` | 手順1で控えた **anon / public キー** |

※追加する際は、**Production**, **Preview**, **Development** すべてにチェックが入った状態で **Save** してください。

## 手順 3: デプロイの反映

設定完了後、新しいプログラムがデプロイされると自動的にデータベース連携が有効になります。
（今回は私がプログラムを更新してプッシュするため、自動的に反映されます）

---

### ※設定を行わなかった場合
設定を行わなくても、プログラムは**自動的にデータベース保存をスキップしてスプレッドシートへの保存のみ**を行います。エラーは発生しませんのでご安心ください。
